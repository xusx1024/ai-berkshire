**毛利率下行的结构性原因：**

FY2022毛利率75.9%是峰值（当时商品销售业务已基本退出，纯平台模式）。此后毛利率逐年下滑，核心原因是Temu全托管/半托管模式涉及仓储、物流、履约等重资产成本，拉低了整体毛利率。国内"千亿扶持"降佣也在压缩毛利。

**经营利润率仍然健康**：即使在Q1 2026，18.4%的经营利润率仍高于绝大多数电商同行（京东约3%，亚马逊约6%）。如果剔除Q1 2026非经营性亏损，调整后净利率约16.8%。

**ROE下行的杜邦分解：**

**数据来源：** SEC 20-F年报现金流量表、Stockopedia现金流数据交叉验证。

**关键发现：**

1. **自由现金流持续大于净利润**：FCF/净利润比率长期超过100%，FY2021高达328%。这说明拼多多的盈利质量极高——利润是"实打实"的现金，而非会计利润。巴菲特最看重的指标之一。

2. **极低的资本开支**：CapEx占营收比例长期低于1%（FY2023仅0.02%），是极致轻资产模式的体现。对比京东CapEx占营收约4-5%，亚马逊约10-12%。

3. **FY2025 CFO首次下降**：经营性现金流从FY2024的1219亿降至FY2025的1069亿（-12%），与净利润降幅一致。反映Temu供应链投资和商家补贴增加。

4. **投资性现金流的"假象"**：FY2024投资性现金流流出1184亿，看似巨额资本支出，实际主要是短期理财产品的买入。真正的固定资产支出仅10亿。

> 但另一方面：三年1000亿元供应链投资计划（年均~333亿）将显著改变拼多多"极致轻资产"的特征。如果这些投入的回报率低于核心业务的40%+ ROIC，将是一个值得警惕的信号。

### 6.5 反向DCF：当前股价隐含什么增长预期？

**方法**：假设WACC 12%（中概股风险溢价）、永续增长率3%、终端FCF利润率15%，反推当前$80股价隐含的营收增速。

**计算过程：**

```
已知：
- 当前市值 = $1,120亿
- 净现金 = $598亿
- 隐含企业价值 = $522亿
- FY2025 FCF ≈ $145亿
- WACC = 12%, 终端增长率 = 3%

终端价值倍数 = 1/(WACC - g) = 1/(12%-3%) = 11.1x

如果假设5年后达到稳态：
- 5年后FCF需要 = EV x (1+WACC)^5 / 终端倍数
- $522亿 x (1.12)^5 / 11.1 = $522亿 x 1.762 / 11.1 = $82.9亿

当前FCF $145亿 -> 5年后$82.9亿 = 年化下降约-11%
```

**结论：当前$80股价隐含的假设是FCF未来5年每年下降约11%**——市场定价假设拼多多的自由现金流将从每年$145亿下降到5年后的约$83亿。

这意味着市场预期：
- Temu的巨额投入将持续侵蚀利润
- 国内业务将停滞甚至萎缩
- 千亿投资计划的回报为零

**如果拼多多只是维持当前FCF水平不增长**，合理估值应为：$145亿/9% x 折现 + $598亿净现金 ≈ **~$180/ADS**——是当前股价的2.25倍。

### 8.2 净现金保护

即使完全忽略拼多多的经营业务，仅看净现金：
- 可自由支配净现金 ~$344亿 / 14.0亿ADS = **$24.6/ADS**
- 总净现金 ~$598亿 / 14.0亿ADS = **$42.7/ADS**
- 当前股价$80中，约**53%是现金支撑**（$42.7/$80）
- 市场对核心业务的隐含定价仅 $80 - $42.7 = **$37.3/ADS**
- 核心业务隐含PE = $37.3 x 14.0亿 / ~$137亿TTM净利润 ≈ **3.8x**

**市场在以3.8倍PE为拼多多的核心业务定价**——几乎是"清算估值"水平，意味着市场认为这个年赚近千亿的业务只值其4年利润。

### 8.3 安全边际的"兑现条件"

安全边际是数学计算的结果，能否兑现取决于催化剂：

**有利催化剂：**
- Temu扭亏为盈（分析师预期2027-2028年）
- 启动分红或回购（概率低但影响巨大）
- 中概股政策缓和
- 国内消费复苏带动广告收入恢复增长

**不利因素（价值陷阱风险）：**
- 管理层持续不回馈股东，低估值长期化
- 利润继续下降（管理层已暗示）
- 欧美关税/监管加码（DSA调查、美国关税）
- VIE结构的法律风险

> 格雷厄姆："市场短期是投票器，长期是称重器。"但对一家不分红的公司，"称重器"需要的时间可能远超投资者的耐心。

### 9.2 估值评分

**★★★★（4星 / 5星）—— 显著低估，但催化剂不确定**

**核心判断：**

当前$80的股价是一个**显著低估的价格**。无论用哪种估值方法——DCF、SOTP、可比公司、反向DCF——都指向至少$120-150的内在价值区间，安全边际50-90%。扣除净现金后，核心业务仅以3.8x PE定价，接近"永久衰退"假设。

**未给5星的原因：**

1. **资本配置的不确定性**——4361亿现金不回馈股东，ROE被持续稀释。如果管理层持续不回购，低估值可能长期存在。
2. **利润趋势向下**——管理层明确"利润将继续下降"，千亿投入的回报高度不确定。
3. **VIE+中概股结构性折价**——不是市场"犯错"，而是真实的结构风险。

**给4星而非3星的原因：**

1. 即使在悲观假设下，当前股价也接近内在价值底线（$85）——**下行空间有限**。
2. 经营层面实际在改善（Q1经营利润+22%），净利润下降是非经营性因素。
3. 净现金$598亿提供了极强的安全垫——占市值53%。
4. FCF收益率13%意味着即使不增长，仅靠现金流回报也相当可观。
