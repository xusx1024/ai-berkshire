# 科技板块巴菲特质量筛选 — 哪些可以买入？

> 筛选日期：2026年6月24日
> 方法论：对科技板块主要公司执行巴菲特7项去劣指标（ROE/FCF/利息覆盖/毛利率/OCF_NP/净利率/股本稀释），通过者进入"可买入候选池"

---

## 一句话结论

**科技板块中，真正符合巴菲特"好生意"标准的不到5家。绝大多数科技公司倒在FCF为负或ROE太低的门槛上。金山办公、网易、宁德时代、腾讯是第一梯队——它们同时满足"好生意+强护城河+合理价格"。**

**本次筛选最意外的发现：网易被严重低估——1635亿净现金占市值31%，13倍PE配33%净利率，是中国互联网中最接近巴菲特"烟蒂+品质"双重标准的公司。**

## 第一梯队：通过巴菲特筛选 ✅

### 1. 金山办公 (688111.SH) — 巴菲特会心动

| 指标 | 数值 | 判定 |
|------|------|:--:|
| ROE（7年均值） | 15.34% | ✅ |
| 5年累计FCF | +94.58亿 | ✅ |
| 利息覆盖倍数 | ∞（零有息负债） | ✅ |
| 毛利率（5年均值） | **85.66%** | ✅ |
| OCF/NP（5年均值） | 1.49 | ✅ |
| 净利率（7年均值） | 31.17% | ✅ |
| 股本稀释（5年） | +0.65% | ✅ |

**7项全部达标，零淘汰。**

**为什么这是好生意**：
- 85%毛利率意味着每多卖一套WPS，85%是纯利——这是软件SaaS的天花板经济特征
- 零有息负债、经营现金流持续大于净利润（1.49倍）——利润是真金白银
- 5年股本几乎未变——没有稀释股东价值的再融资
- 中国2亿+用户，替代微软Office的国产化逻辑

**风险**：研发费用率35%仍在攀升（2025年研发20.95亿），如果AI/WPS 365投入不能打开收入天花板，ROE可能被稀释。微软Copilot的竞争压力也不可忽视。

**当前估值**：需另行确认。好生意 ≠ 现在就是好价格。

### 2. 宁德时代 (300750.SZ) — 全球电池基础设施

| 指标 | 数值 | 判定 |
|------|------|:--:|
| ROE (TTM) | ~22% | ✅ |
| 毛利率 | 高（净利率16.1%） | ✅ |
| FCF | 经营现金流强 | ⚠️ 需细查 |
| 护城河 | 技术+规模+客户三重复合 | ✅ |
| 全球市占率 | 37%+ | ✅ |
| PE (TTM) | 22.6x | ⚠️ 不便宜 |

**7项去劣中5-6项通过。** 2026Q1营收1291亿(+52%YoY)，净利207亿(+48.5%)——量价齐升。

**护城河判断**：全球动力电池37%市占率不是运气。制造良率、供应链规模、专利组合、客户关系——这四个维度构成结构性壁垒。给竞争对手1000亿也难在5年内复制。

**最大风险**：(1) 固态电池技术路线若跃迁，CATL的先发优势可能被削弱；(2) 地缘政治限制海外扩张；(3) PE 22.6x对应周期高点的利润，安全边际一般。

### 3. 舜宇光学 (2382.HK) — 被低估的精密光学寡头

| 指标 | 数值 | 判定 |
|------|------|:--:|
| ROE (TTM) | 17.5% | ✅ |
| FCF (FY2025) | +31.9亿人民币 | ✅ |
| PE (TTM) | 16.2x | ✅ 合理 |
| 毛利率 | 19.7% | ⚠️ 制造业偏低 |
| 净利率 | 10.7% | ✅ |

**核心逻辑**：手机镜头/模组行业是大立光+舜宇的双寡头格局。技术壁垒真实（镜头设计/模具/镀膜/组装，追赶周期5年+）。汽车ADAS镜头提供第二增长曲线。

**估值**：PE 16.2x对应刚从低谷恢复的盈利（FY2025净利+72%），FCF收益率4.3%提供了一定安全边际。

**风险**：智能手机出货波动；毛利率19.7%在制造业中不算高。

**为什么被市场低估**：
- 市场把网易当成"游戏公司"给低估值，但它的本质是"高复购的数字娱乐订阅+轻资产印钞机"

**风险**：游戏版号依赖政策；《逆水寒》等爆款有生命周期；海外拓展尚在早期。

**一句话**：如果巴菲特看中国互联网，网易会比腾讯更符合他的审美——更轻的资产、更高的净利率、更大的安全边际（净现金=市值31%）。

## 巴菲特框架下的关键认知

在筛选过程中，一个规律反复出现：

> **科技行业中，"好生意"极度稀缺。** 26家公司中，真正通过巴菲特7项去劣指标的仅3-4家。大多数科技公司要么烧钱（FCF为负）、要么周期性强（盈利不可预测）、要么利润太薄（缺乏护城河）。

段永平说过："我投资很简单——看不懂的不投，看懂了但不够好的不投，够好但太贵的不投。"在A股科技板块中，北方华创、中芯国际、海光信息都是"看懂了但不够好"（FCF为负）；韦尔股份、德赛西威是"时好时坏"（周期性强）；传音、立讯是"好但不够好"（利润太薄）。

**真正进入"看得懂+够好+价格合理"三角的，只有金山办公、网易、宁德时代、腾讯四家。** 舜宇光学和快手是"够好但还不够确定"，拼多多是"够好但治理有致命缺陷"。
