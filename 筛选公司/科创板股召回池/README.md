## 核心发现

1. **半导体设备是最优质赛道**——7家通过，"卖铲人"模式经去劣检验最扎实
2. **高毛利是好公司的共性**——通过公司毛利率中位数~45%，对比全部100家中位数~25%
3. **科创板整体盈利质量堪忧**——56%的公司因亏损被淘汰，"科技成长"≠"好生意"
4. **值得长期跟踪的边界公司**——海光信息、拓荆科技、中国通号等18家需等待更多财务数据验证
