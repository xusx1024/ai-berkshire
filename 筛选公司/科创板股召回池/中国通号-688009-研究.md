# 中国通号（688009.SH）深度投资研究

> 研究日期：2026年5月17日
> 数据截止：2025年年报 + 2026Q1季报

---

## 一、商业模式与竞争壁垒

### 1.1 主营业务拆解（2024年收入结构）

| 业务板块 | 营收（亿元） | 占比 | 同比增速 | 毛利率 |
|---------|------------|------|---------|--------|
| 铁路领域 | 188.90 | 58% | -1.86% | 29.95% |
| 城轨领域 | 82.20 | 25% | -3.58% | 33.47% |
| 海外业务 | 24.41 | 7% | +64.20% | 24.61% |
| 工程总承包 | 26.89 | 8% | -64.96% | 11.98% |

来源：[2024年报点评-新浪财经](https://finance.sina.com.cn/roll/2025-03-31/doc-inerqmnh3624987.shtml)

**商业模式本质**：覆盖轨道交通控制系统"设计咨询→设备制造→系统集成→运维服务"全产业链。铁路+城轨核心业务占比83%，是典型的"国家基建配套型"企业。

### 1.2 行业地位与市占率

- **高铁控制系统**：国内市占率约60%，绝对龙头
- **城轨信号系统**：国内市占率约48.5%（含子公司卡斯柯、通号城轨），第一名
- **全球排名**：全球最大轨道交通控制系统供应商之一
- 拥有5500+项注册专利，列控核心技术100%自主化

来源：[腾讯新闻-交通强国报告](https://news.qq.com/rain/a/20230428A01MRE00)

### 1.3 护城河分析

**强护城河因素**：
- **技术壁垒极高**：轨道交通信号系统关乎行车安全，认证周期长（3-5年），新玩家几乎无法进入
- **客户粘性强**：国铁集团/地铁公司是核心客户，替换成本极高
- **标准制定者**：参与国内轨道交通信号技术标准制定
- **规模效应**：研发投入超20亿/年，中小企业无力跟进

**护城河弱化因素**：
- 国企体制，市场化激励不足
- 城轨领域竞争者逐渐增多（交控科技、和利时等蚕食份额）
- 依赖政府铁路/城轨投资计划，议价权有限

## 二、财务分析

### 2.1 盈利能力

| 营收（亿） | 326.44 | 346.77 | 恢复增长+6.2% |
| 归母净利（亿） | 34.95 | 36.86 | +5.5% |
| 毛利率 | 29.22% | 28.88% | 略降 |
| 净利率 | 12.53% | 12.29% | 略降 |
| ROIC | 6.78% | ~7% | 偏低 |
| ROE（半年度年化） | ~7.9% | ~6.8% | 下行 |

来源：[2025年报简析-腾讯新闻](https://news.qq.com/rain/a/20260326A025GZ00)、[2024年报简析-腾讯新闻](https://news.qq.com/rain/a/20250330A01N5Q00)

**犀利点评**：ROE年均约9%（历史），近两年持续下滑至7%左右。作为重资产国企，资本回报率平庸。毛利率29%看似不错，但扣除工程总承包后核心业务毛利率30-33%，净利率仅12%——管理费用率7.3%+研发费用率5.9%吃掉大量利润。

### 2.2 现金流质量

| 指标 | 2024年 | 2025年 |
|------|--------|--------|
| 经营现金流净额（亿） | 52.36 | 约-13（每股-0.12元） |
| OCF/净利润 | 150% | 负值 |
| 应收账款/净利润 | 771.83% | 678.21% |
| 收现比（5年均值） | 0.94 | — |

来源：[2024年报-新浪财经](https://finance.sina.com.cn/roll/2025-03-31/doc-inerqmnh3624987.shtml)、[2025年报简析](https://news.qq.com/rain/a/20260326A025GZ00)

**犀利点评**：现金流是这家公司最大的硬伤。2025年经营现金流转负，应收账款长期高企（接近净利润的7倍）。5年平均收现比0.94，说明每赚100元账面利润只收回94元现金。作为一家"赚纸面利润"的企业，自由现金流长期不理想，分红能力受限。核心原因是客户（国铁集团、各地铁公司）回款周期长，属于典型的"大甲方小乙方"格局。

负债率58%对央企而言尚可，但经营性现金流覆盖流动负债的比例极低（仅11-15%），偿债主要靠借新还旧而非经营造血。

## 三、成长性

### 3.1 收入增长驱动力

1. **铁路存量改造市场**：全国超1万公里高铁进入大修改造期，信号系统升级是刚需
2. **海外市场爆发**：2024年海外营收+64%，2025H1海外营收+55.9%，已成为核心增长极
3. **低空经济新赛道**：构建"低空空域管控+无人机制造+服务运营"布局，2025H1签约0.3亿元（体量极小）
4. **城轨后市场**：既有线路维护升级需求

### 3.2 在手订单情况

- **截至2025年6月底**：在手订单1175.63亿元
- 相当于2024年营收的**3.6倍**，保障至少3年正常经营
- 2025年上半年新签合同175.68亿元

来源：[新浪财经-在手订单报道](https://finance.sina.com.cn/roll/2025-09-29/doc-infscaav8404282.shtml)

### 3.3 海外拓展进度

- 2024年海外营收24.41亿元（+64.2%），占比7%
- 2025H1海外营收9.44亿元（+55.9%）
- 主要覆盖"一带一路"沿线国家铁路信号系统出口
- 海外毛利率24.61%，低于国内，但增速快、空间大

来源：[银河证券研报](https://bigquant.com/square/paper/9e7306d6-6b11-4fe3-8ab2-0a607b2b2c6d)

**犀利点评**：增长故事有逻辑但缺乏爆发力。铁路改造是"维持型"而非"增长型"需求；低空经济签约0.3亿对346亿营收而言微不足道，至少3-5年内无法贡献实质收入；海外业务增速亮眼但基数小，且毛利率低于国内。整体判断：低个位数增长的成熟企业，非高成长股。

## 四、估值与风险

### 4.1 当前估值水平

| 指标 | 数值 |
|------|------|
| 最新股价 | ~5.07元（2026年5月） |
| 总股本 | ~105.9亿股 |
| 总市值 | ~537亿元 |
| 2025年EPS | 0.34元 |
| PE TTM | ~14.9倍 |
| PB | ~1.13倍 |
| 股息率 | ~2.5%（估算） |

来源：[腾讯证券](https://gu.qq.com/sh688009)、[东方财富](https://data.eastmoney.com/stockdata/688009.html)

**估值判断**：15倍PE对应7%的ROE和个位数增长，谈不上便宜。PB仅1.1倍反映市场对其资产质量（大量应收账款）的担忧。与同类国企（中国中车PE 20x+）相比估值略低，但盈利质量也更差。

### 4.2 主要风险点

1. **应收账款风险**：应收账款占净利润的7倍，若出现坏账集中暴露将严重侵蚀利润
2. **铁路投资周期风险**：国家铁路投资若大幅下滑，公司营收将承压
3. **现金流恶化风险**：2025年经营现金流已转负，若持续恶化将影响分红和再投资能力
4. **城轨竞争加剧**：交控科技等竞争者持续蚕食城轨份额
5. **国企治理效率**：ROE持续走低反映资本配置效率不高
6. **2026Q1业绩下滑**：营收-3.5%、净利润-7.2%，增长动能减弱

### 4.3 投资结论

| 维度 | 评分 | 说明 |
|------|------|------|
| 商业模式 | ★★★★ | 轨交信号龙头，壁垒高、地位稳 |
| 盈利能力 | ★★☆ | ROE仅7%，资本回报平庸 |
| 现金流质量 | ★★ | 长期"纸面利润"，应收账款巨大 |
| 成长性 | ★★☆ | 海外有亮点但整体低增长 |
| 估值吸引力 | ★★★ | 15x PE不贵但也不便宜 |
| **综合评分** | **★★☆** | **2.5星/5星** |

### 最终判断

中国通号是一家"好生意、差公司"的典型案例。轨交信号系统本身壁垒极高、市占率稳固，但国企体制下资本配置效率低下、现金流质量差、增长乏力。15倍PE对应7%的ROE和个位数增长，没有明显的安全边际。

**不适合作为核心持仓**。除非出现以下情况才值得重新评估：
- PE降至10倍以下（股价3.4元）
- 海外业务占比突破20%且毛利率改善
- 现金流质量实质性改善（收现比持续>1）
